## Usage

```text
usage: crack.sh [-z ZIP] [-d DICT] [-v]

optional arguments:
  -z ZIP, --zip ZIP     path to zip file
  -d DICT, --dict DICT  path to dictionary file
  -v, --verbose         verbose
```
